# Assignment
Assignment 1
## Title
How the Internet Delivers a Web Page Using TCP/IP Protocol Stack
## Objective
To understand how the Internet delivers a web page using TCP/IP, DNS, HTTP, and TCP
## Solution
The browser uses DNS to find the server's IP address, sends an HTTP request through the TCP/IP protocol stack, and receives the webpage reliably from the server.
## Output
The requested YouTube webpage is successfully displayed in the user's browser after reliable data transmission.
## Conclusion
Application-layer protocols such as DNS, HTTP, and TCP work together to provide reliable and efficient Internet communication.
